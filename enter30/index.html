一、Assembly

Kit Includes

  • Shell *1
  • PCB *1
  • Foam *1
  • Battery *1
  • USB small board (pre-installed in the bottom shell)
  • Strap *1
  • Accessory pack

Items to Prepare

  • Scissors
  • Tweezers
  • Several switches
  • 2U satellite switches *3

Bottom Shell Processing

Take out the foam strips from the accessory pack, stick 4 strips on the top and bottom protrusions, and 1 strip on each side protrusion (need to cut them open yourself), as shown in the image below.

Top Shell Processing (Optional)

Stick the corresponding number of foam strips in the grooves on the top shell.

Insert Switches

Stack the positioning plate, foam, and PCB (in that order), insert the switches, and install the satellite switches.

Connect Wires

Flip the assembled inner shell, connect the USB cable and battery cable, the interface order is as shown in the image below.

Assemble Top and Bottom Shells

After connecting the wires, lay the inner shell flat on the bottom shell, take out the top shell, and press it together with the bottom shell.
Take out the screws and screwdriver, flip the keyboard over, and tighten the screws.

二、Set Up Bluetooth

The Vial software requires the version from the link below; other versions may cause Bluetooth-related key values to be garbled.

Software Download

Click the link below to access the download page.
Vial https://drive.google.com/drive/folders/1WWxKntgUshDlofq6pLYYpML3d8jF486k?usp=sharing

Software Operation

Connect the keyboard to the computer and open the Vial software.
Select the keyboard layer where you want to place the Bluetooth configuration, then select the Quantum tab below, and find the last 6 key values.
  • BLE0：Bluetooth Channel 1
  • BLE1：Bluetooth Channel 2
  • BLE2：Bluetooth Channel 3
  • BLE DEL；Delete the current channel
  • USB MODE: Switch the keyboard to USB mode
  • RADIO MODE: Switch the keyboard to 2.4G mode
Return to layer 0, set a layer switch key, for example, to switch to the second layer to trigger Bluetooth.

Bluetooth Pairing

Unplug the keyboard USB cable, turn on the battery switch, up is on, down is off.
Switch to the Bluetooth layer, trigger the BLE0 key value, at this point the keyboard is in Bluetooth broadcasting mode, open the computer Bluetooth settings, and select to add a Bluetooth device.
Click on Bluetooth.
Click on ENTER30_BLE_O in the Bluetooth interface.
After the system prompts that it is ready, click Complete.

Bluetooth Instructions

  1. The Enter30 supports a total of 3 Bluetooth channels, corresponding Bluetooth device names are: ENTER30_BLE_0, ENTER30_BLE_1, and ENTER30_BLE_2.
  2. To switch from wireless mode to wired mode, after plugging in the cable, you need to trigger the USB MODE key value.
  3. To switch from wired mode to wireless mode, you need to trigger the corresponding channel key value.
  4. If the current channel needs to bind a new device, you need to trigger the BLE DEL key value.
